# GPU Computing project
## Description
GPU computing project for the master degree in Computer Science at UniTn (University of Trento).

It consists in the implementation and optimization of the Spmv (Sparse Matrix Multiplication Vector) using Cuda.
<img width="660" height="338" alt="image" src="https://github.com/user-attachments/assets/30ec115b-b493-4e86-a45b-12ac267d72d9" />


### Comparison with CPU
In the [first deliverable](deliverable1), we compare the performance of various cpu 
implementation with a basic GPU implementation.

### GPU implementation optimization
In the [second deliverable](deliverable2), we try to optimize the GPU implementation by changing the 
access pattern, trying to reduce misses, configuring right parameter,... (more details in the link).
